## Projeto

**Descrição:** O meu objetivo inicial para o projeto é desenvolver um protótipo de um jogo para a web. Por enquanto, a ideia é construir um aplicativo de calendário em que o objetivo é realizar tarefas dentro de um limite de tempo. A interface será um calendário, no qual "compromissos" surgirão aleatóriamente pelo mês. Cada compromisso tem uma data de entrega máxima (o dia em que ele aparece no calendário) e interagir com o compromisso revela uma tarefa a ser feita. O tempo passa aos poucos e o jogador deve completar cada tarefa antes que o dia de entrega acabe. Deixar de fazer tarefas faz o jogador perder pontos.
